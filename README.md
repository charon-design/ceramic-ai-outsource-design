# AI非遗醴陵陶瓷文创外包设计平台
## 一、项目背景
醴陵釉下彩陶瓷本地中小工厂、文旅礼品店存在设计外包痛点：人工绘制纹样周期长、调色不符合印刷标准、批量出图成本高。本项目打造轻量化AI外包设计工具，为商家提供纹样生成、图像修复、印刷标准化处理一站式外包服务。

## 二、仓库代码文件功能说明
### 1. ceramic_process.py
批量图像处理Python脚本，是项目核心技术模块：
- 对瓷器实拍图自动降噪、高清修复、优化釉下彩色彩饱和度；
- 一键转换图片为瓷厂印刷专用CMYK色彩模式，避免烧制色差；
- 支持文件夹批量自动化处理，模拟大批量外包修图订单交付流程。

### 2. index.html
静态前端演示页面，用于比赛录屏展示外包接单流程：
页面划分需求提交区、AI纹样预览区，可录入醴陵瓷纹样关键词、上传瓷器底图，模拟AI生成纹样、导出印刷商用成品的完整外包交付环节，无需服务器，浏览器直接打开运行。

## 三、代码联动完整外包业务流程
商家在index.html前端页面提交陶瓷纹样设计需求与瓷器底图
→ AI自动生成适配釉下彩工艺的纹样初稿
→ 将原图、初稿导入ceramic_process.py批量脚本完成画质、色彩标准化处理
→ 输出符合印刷标准的成品图片，完成外包交付

## 四、项目配套素材
仓库配套自主设计「株小瓷」醴陵陶瓷IP、釉下山水花鸟纹样、茶具礼盒包装效果图，作为外包服务成品案例，佐证项目落地可行性。

## 五、项目优势
1. 垂直产业适配：代码逻辑针对醴陵陶瓷印刷工艺定制，区别于通用修图工具；
2. 轻量化易演示：无复杂部署，普通电脑即可运行两套代码，适合创客赛道展示；
3. 外包流程闭环：前端接单页面+批量处理脚本完整复刻文创外包全业务链路。
